# Commands used

## First initialize our directory as npm package:

`npm init -y`

## To Install Cypress

`npm install cypress --save-dev`

## To run:
`npx cypress open` or `./node_modules/.bin/cypress open`

### or add the following script in package.json

```
 {
  "scripts": {
    "cypress:open": "cypress open"
  }
}
```
